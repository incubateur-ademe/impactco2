Le script iframe.js détecte automatiquement certains IDs historiques dans le DOM et décide du type de widget à charger en fonction de l'ID trouvé. Chaque section ci-dessous utilise un ID différent. Seul le premier est activé par défaut (les autres sont commentés car le script ne détecte que le premier trouvé).

#mon-impact-transport

Cas spécial : cet ID génère un bouton de redirection, pas un iframe. Le script détecte cet ID en priorité.

#impact-co2

ID générique qui charge le comparateur par défaut.

#impact-livraison

ID spécifique au module livraison.

#datagir-mon-convertisseur-co2

Ancien ID Datagir pour le convertisseur CO2. Charge le comparateur.

#datagir-impact-co2

Ancien ID Datagir générique. Charge le comparateur.

#ecolab-transport

Ancien ID Ecolab pour le transport.

#datagir-teletravail

Ancien ID pour le télétravail. Redirige vers le module teletravail.
